# GPO Navigator — Downloads & Updates

Release downloads and the auto-update manifest for **GPO Navigator**, a
local-first Windows desktop app for searching large GPO (Group Purchasing
Organization) membership datasets.

> This repository hosts **downloads and updates only** — the application source
> is maintained privately.

## ⬇️ Download

**[➡ Download the latest installer](../../releases/latest)**

On the latest release, download `GPO Navigator_<version>_x64-setup.exe` and run it.

## Installing & updating

1. Run the `…-setup.exe` **once** to install.
2. After that, the app **updates itself** — on launch it checks here for a newer
   version and offers a one-click update (download → install → relaunch). No more
   downloading installers by hand.

## What it does

GPO Navigator replaces manual lookups across multiple spreadsheets. Import
Excel/CSV membership exports and search them instantly by **customer name,
address, GPO ID, GLN, ZIP, or parent** — with Premier tier classification,
GI/eyeMAX price-list references, conflict detection, and bulk lookup. Everything
runs **locally**; no internet connection or sign-in is required for day-to-day use.

Highlights:

- Fast ranked search over millions of rows, with exact-match highlighting
- Keyboard-driven: arrow keys through results, `/` or `Ctrl+F` to search
- Sortable, resizable, reorderable result columns
- Copyable email-response templates and price-list codes
- Compressed local backups + restore
- Automatic in-app updates
